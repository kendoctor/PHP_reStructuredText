Text\RestructuedはreStructuredText形式のテキストを 解析する為のパーサーキットです。 Renderは現状用意されておりませんので、自前で実装する必要があります。

How to use this library?

使い方についてはtestディレクトリの中をみつつチャレンジしてみてください。

Purpose

現状Machine側の実装はかなり適当でparseできないものが多くあるかと思います。 parseできないものがあればchobi_eまで教えてもらえれば対応できる限り 対応しようかな、と思います。

今のところ主にSymfony2のドキュメントをparseするのが一番の目的で。作っていく過程で parserの精度が実用的であれば標準化するつもりです。今後は主にgithubで開発をしていくので、参加したい方はガンガンpullリクエストを 送ってください。

対応状況

今のところの対応状況はこんな感じ:: 

[*]章立ての構造
  =======  タイトル
  =======  
  サブタイトル
  -----------  
[*]段落
  テキスト
[*]記号付きリスト
  something
    lorem ipsum dolar sit amet
  anything
[]番号付きリスト
  1. something
      lorem ipsum dolar sit amet
[]autoincrementリスト
  #. moemoe
  
[]定義リスト
  左揃えテキスト
    インデントテキスト（空行なし）
[-]整形済みブロック
  ::
  
    インデントおわりまで
 
  something::
  
    インデント終わりまで（上記の場合はコロンがひとつになる）
 
  > text
[*]ラインブロック
  装飾の不要なリスト

[*]引用
  インデントするだけ
[*]Doctestブロック
  something

[*]区切り線
  ------------とか
[*]フィールドリスト
  題名    インデントテキスト
    (インデント終わりまで続く)
[] 拡張系の実装
  [*]コメント
--未対応
[]表（面倒くさいのでやらない。気が向いたら）
  ・グリッドテーブル
  ・シンプルテーブル
  
[]オプションリスト
  -a            command-line option "a"
  -b file       options can have arguments
                and long descriptions
  --long        options can be long also
  --input=file  long options can also have arguments
  /V            DOS/VMS-style options too
